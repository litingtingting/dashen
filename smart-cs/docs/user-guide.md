# 智能客服系统 - 用户文档

## 👋 欢迎使用智能客服系统

智能客服系统帮助您的店铺/企业快速接入 AI 智能客服，支持微信公众号、淘宝店铺、抖音等多个平台，7×24 小时自动回复客户咨询。

---

## 🚀 快速开始

### 第一步：申请接入

1. 访问企业对接页面：`https://your-domain.com/join`
2. 填写企业信息
3. 选择需要接入的平台（微信/淘宝/抖音）
4. 选择套餐（体验版/标准版/专业版/企业版）
5. 提交审核

### 第二步：等待审核

- 丽斯团队会在 **1-3 个工作日** 内审核
- 审核通过后会通过短信/邮件联系您
- 提供对接所需的账号和权限

### 第三步：配置对接

根据接入平台不同，需要准备：

#### 微信公众号

| 材料 | 说明 |
|------|------|
| 公众号 AppID | 登录公众号后台获取 |
| 公众号 AppSecret | 公众号后台 → 开发 → 基本配置 |
| 服务器域名 | 我们提供，您配置到公众号后台 |

#### 淘宝店铺

| 材料 | 说明 |
|------|------|
| 淘宝店铺账号 | 正常经营的店铺 |
| 子账号权限 | 开通客服权限 |

#### 抖音企业号

| 材料 | 说明 |
|------|------|
| 抖音企业号 | 蓝 V 认证企业号 |
| 管理员权限 | 可以授权第三方应用 |

### 第四步：测试上线

- 我们会提供测试账号
- 确认回复内容和人设风格
- 正式上线使用

## 🎭 人设配置指南

### 什么是人设？

人设是您的 AI 客服的"性格"和"说话风格"，不同店铺可以有不同的客服人设。

### 人设配置项

| 配置项 | 说明 | 示例 |
|--------|------|------|
| 人设名称 | 客服的昵称 | "贴心小助手" |
| 语气风格 | 回复的整体风格 | "温暖亲切" |
| 问候语 | 客户首次咨询的欢迎语 | "亲～欢迎来到小店！💕" |
| 常用表情 | 喜欢用的表情符号 | 💕😊🌹 |
| 称呼方式 | 如何称呼客户 | "亲"、"宝子"、"您" |

### 不同平台推荐风格

#### 淘宝店铺

```
人设名称：贴心小管家
语气风格：热情亲切
问候语：亲～欢迎光临小店！有什么可以帮您的吗？💕
特点：多用"亲"、"包邮"、"优惠"等电商用语
```

#### 微信公众号

```
人设名称：小助手
语气风格：专业友好
问候语：您好！欢迎关注 XX 公众号，有任何问题都可以问我哦～
特点：专业但不失亲切，适度使用表情
```

#### 小红书

```
人设名称：种草小姐姐
语气风格：姐妹风
问候语：宝子！来啦～今天想了解什么呀？💖
特点：真实分享感，多用网络热词
```

## ❓ 常见问题

### Q1: AI 回复准确吗？

A: AI 基于您的人设配置和知识库进行回复，准确率约 85-95%。建议：
- 配置详细的人设和常用语
- 定期查看对话记录，优化人设
- 复杂问题可转人工客服

### Q2: 可以完全替代人工客服吗？

A: 目前建议 AI+ 人工结合：
- AI 处理常见问题（80%）
- 人工处理复杂问题（20%）
- 非工作时间 AI 值守

### Q3: 对话记录保存多久？

A: 默认保存 7 天，7 天后自动清理。专业版可以标记重要对话永久保存。

### Q4: 如何保证回复合规？

A: 系统有多重保障：
- AI 输出自动过滤敏感词
- 符合广告法和平台规则
- 可配置自定义过滤词库
- 可疑内容标记人工审核

### Q5: 可以同时接入多个平台吗？

A: 可以！标准版支持 2 个平台，专业版支持 5 个平台，企业版不限。

### Q6: 套餐用超了怎么办？

A: 超出每日上限后：
- 体验版/标准版：暂停 AI 回复，转人工
- 专业版/企业版：按量计费（¥0.1/次）
- 可升级套餐

### Q7: 如何申请发票？

A: 联系丽斯团队客服，提供：
- 企业抬头
- 税号
- 邮箱地址
- 发票类型（普票/专票）
