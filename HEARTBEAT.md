## 📊 检查项

### 服务器健康
- CPU 使用率 < 80%
- 内存使用率 < 80%
- 磁盘剩余空间 > 20%

### API 状态
- 百炼 API 可调用
- 响应时间 < 5 秒
- 错误率 < 1%

### 代码质量
- Git 无未提交变更
- 无严重 bug
- 测试覆盖率 > 70%

## 🎯 Heartbeat 触发逻辑

**当收到 heartbeat 消息时**：
1. 读取本文件
2. 检查当前时间
3. 如果是每日 8:00-9:00 → 执行每日汇报
4. 如果是周日 20:00-21:00 → 执行周报
5. 如果是每月 1 号 9:00-10:00 → 执行月检
6. 其他时间 → 回复 HEARTBEAT_OK
